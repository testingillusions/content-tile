Content Card Plugin Test

Example Shortcodes

1. Basic card with links only (no access restriction)

[content_card 
  title="Plan Comparison Tool" 
  link1_text="Plan Comparison Tool" 
  link1_url="/plans"
  link2_text="PCT FAQ" 
  link2_url="/faq"
  link3_text="PCT Helpful Hints" 
  link3_url="/hints"
  paragraph_text="Compare 2-4 plans side-by-side with our interactive Medical Plan Comparison Tool"
]

2. Card with new button system (access restricted)

[content_card 
  title="Premium Content" 
  access_group_ids="1,2"
  link1_text="Resource 1" 
  link1_url="/resource1"
  link2_text="Resource 2" 
  link2_url="/resource2"
  paragraph_text="Access premium content with your subscription"
  button1_text="Upgrade Now" 
  button1_url="/upgrade"
  button2_text="Schedule Demo" 
  button2_url="/demo"
  button3_text="Learn More"
  button3_url="/info"
]

3. Card with empty links (should show blank links)

[content_card 
  title="Minimal Card" 
  link1_text="Available Link" 
  link1_url="/available"
  link2_text="" 
  link2_url=""
  link3_text="Another Link" 
  link3_url=""

]

4. Legacy compatibility test

[content_card 
  title="Legacy Card" 
  access_group_ids="1"
  upgrade_text="Old Upgrade Button" 
  upgrade_url="/legacy-upgrade"
  demo_text="Old Demo Button" 
  demo_url="/legacy-demo"
]

Key Features Implemented:

  • ✅ Links only display when both text AND URL are provided
  • ✅ Added paragraph section at bottom with centered text
  • ✅ Up to 3 overlay buttons with different styling
  • ✅ Buttons only display when both text AND URL are provided
  • ✅ Backward compatibility with legacy upgrade/demo buttons
  • ✅ New button system takes precedence over legacy system
